### SCORING RUBRIC

**Ease of Implementation (1-10 scale)**
- 9-10: No new tools/skills required; plug into existing agentic workflow
- 7-8: Minimal new infrastructure; existing agent framework can adapt
- 5-6: Requires new agent type or minor system expansion
- 3-4: Requires significant new infrastructure or domain expertise
- 1-2: Requires new personnel, legal review, or major system redesign

**Profitability Potential (1-10 scale)**
- 9-10: Revenue path clear; can monetize within 3 months ($100K+ annual)
- 7-8: Revenue path exists; 6-month path to commercialization ($50K+ annual)
- 5-6: Revenue potential but requires partnership or distribution ($25K+ annual)
- 3-4: Ancillary revenue (supports existing products); low direct income
- 1-2: Strategic value only; no direct revenue path

**Agentic Build Feasibility (1-10 scale)**
- 9-10: Can build with existing agent stack; requires task routing + config
- 7-8: Requires new agent type; moderate development effort
- 5-6: Requires custom agent architecture; significant development
- 3-4: Requires novel AI/ML capability not in current stack
- 1-2: Requires external vendor or human domain expertise

**Strategic Alignment with Studio Core (1-10 scale)**
- 9-10: Direct alignment with Sentinel, eBay, Game Archaeology, Ghost Book, Historical Figures
- 7-8: Strong alignment with 2+ existing projects
- 5-6: Supports 1 existing project; tangential to others
- 3-4: Adjacent to studio mission; low overlap
- 1-2: Exploratory; minor strategic value
